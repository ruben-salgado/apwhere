1. ¿Qué entiende por herencia?

Es una herramienta con la que se pueden transferir propiedades y características de un objeto más general a uno más específico, un ejemplo sería un vehículo, que es el elemento padre, con el cual se puede trasladar una persona o un objeto, y el elemento más particular es un automóvil o un avión, en donde ambos poseen sus propias características específicas, pero con ambos se puede llevar cosas de un punto a otro, es decir, comparten la propiedad que es un medio de transporte y la acción del desplazamiento.

2. Define el concepto 'clase' en la programación orientada a objetos

Forma en la cual se pueden representar las cosas tangibles o de entendimiento coloquial y trasladarlas al código para llevar a cabo una tarea específica. La forma en la que se estructuran es usando propiedades y acciones.

3. ¿Qué entiende por polimorfismo?

Cuando se quiere realizar acciones que son distintas o que tienen alguna variante en su flujo y para ello cambian las propiedades o argumentos, pero se conserva el mismo nombre.

SELECT e.NOMBRE AS ESTADO,
    CASE
        WHEN SUM(p.NUMERO_EMPLEADOS) > 0 THEN

11. Describe en breves palabras el patrón de arquitectura de software MVC.

Su estructura se fundamenta en la separación del código haciendo que cada componente pueda trabajar de manera independiente y de ser necesario cambiar su implementación sin afectar al resto de capas.
  • Vista: Es lo que puede visualizar el usuario directamente.
  • Modelo: En esta capa es en dónde se encuentra toda la lógica de negocio, es decir, la manipulación de los datos obtenidos de diferentes fuentes, para tener una estructura adecuada a la solicitud.
  • Controlador: Es quien procesa y formatea la información que verá el usuario. Decide como se van a presentar los datos obtenidos por el modelo y como se van a enviar hacia la vista.

12. Define que es XPATH.

Es un sistema usado para navegar por el árbol de un archivo XML y poder leer la estructura del mismo.

13. Define que es AJAX.

Cuando se requiere que se puedan atender múltiples operaciones al mismo tiempo o incluso cuando solo es una operación, pero que no se desea bloquear al usuario, es cuando se utiliza ajax, que es una tecnología que permite comunicación asíncrona desde el cliente hacia el servidor.

14. Define que es JSON

Un formato en el que se estructura la información de la forma clave - valor, encerrando sus datos entre llaves. Usado ampliamente para el intercambio de datos cliente-servidor, entendiendo que el cliente es quien solicita los datos y éste también puede ser otro servidor o un dispositivo que tiene el usuario final.

15. ¿Qué es JPQL?

Lenguaje que permite de forma rápida realizar consultas a la base de datos mediante anotaciones en las clases de java, evitando así usar consultas directas a la base, sustituyéndolas por consultas más orientadas hacia una lógica entre objetos puramente de java.
